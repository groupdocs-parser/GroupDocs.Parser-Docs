# GroupDocs.Parser - Documentation Source Files

This repository contains Markdown files that are used to generate GroupDocs.Parser static documentation that can be found at <https://docs.groupdocs.com/parser/>

## About GroupDocs.Parser

[GroupDocs.Parser](https://products.groupdocs.com/parser/) is a document processing SDK that enables you to parse documents and extract data such as text, tables, images and barcodes.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official documentation.

- Ensure documentation is always up-to-date with the latest product release.

- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
