# GroupDocs.Parser - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Parser](https://docs.groupdocs.com/parser/) static documentation.

## About GroupDocs.Parser

[GroupDocs.Parser](https://products.groupdocs.com/parser/) is a document parsing SDK that enables you to extract data such as text, tables, images, and barcodes from documents in various formats.

Available for .NET, Java, and Python.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/parser/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
