## Overview

[GroupDocs.Parser for Python via .NET](https://products.groupdocs.com/parser/python-net) does not require any external software or third-party tool to be installed. Just follow one of the ways described in the [Installation]({{< ref "parser/python-net/getting-started/installation" >}}).

## Supported Operating Systems

GroupDocs.Parser for Python via .NET supports any 32-bit or 64-bit operating system where Python is installed including, but not limited to:

### Windows

- Microsoft Windows Server 2003 and later
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows 7, 8, 8.1 (x64, x86)
- Microsoft Windows 10 (x64, x86)
- Microsoft Windows 11 (x64)

### Linux

- Linux (Ubuntu, OpenSUSE, CentOS and others)

### Mac

- Mac OS X

## Supported Frameworks

GroupDocs.Parser for Python via .NET supports the following Python frameworks:

### Python

- Python 3.5, 3.6, 3.7, 3.8, 3.9, 3.10, 3.11, 3.12 (recommended)

## Development Environments

GroupDocs.Parser for Python via .NET can be used with any text editor or IDE that supports Python development, including:

- PyCharm
- Visual Studio Code
- Jupyter Notebook
- Sublime Text
- Atom
- Spyder
- Any other Python-compatible IDE or text editor

## Additional Dependencies

GroupDocs.Parser for Python via .NET automatically installs all required dependencies when installed via pip. No additional packages need to be manually installed.
