**GroupDocs.Parser for Python via .NET** is a Python version of the GroupDocs.Parser API. It provides the ability to extract text, images, metadata, tables, and structured data from over 50 document formats. This powerful API supports parsing documents with templates for structured data extraction, as well as raw and formatted text extraction modes. The library supports popular formats including PDF, Microsoft Office, OpenOffice, images, archives, and more.
